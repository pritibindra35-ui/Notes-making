Code Examples & Formatting

How to Format Code in Your Notes

Example: List Operations in Python
# Problem: Find maximum element in a list
# Algorithm:
# 1. Initialize first element as max
# 2. Compare with each element
# 3. Update max if larger element found

def find_max(numbers):
    """
    Function to find maximum element in a list
    Input: list of numbers
    Output: maximum number
    """
    if not numbers:  # Check for empty list
        return None
    
    max_num = numbers[0]  # Initialize with first element
    
    for num in numbers[1:]:  # Iterate from second element
        if num > max_num:
            max_num = num
    
    return max_num

# Example usage:
my_list = [23, 45, 12, 67, 34, 89, 15]
result = find_max(my_list)
print(f"Maximum number: {result}")

# Output: Maximum number: 89
Note-taking Tips for Code:
  • Always include the problem statement
  • Write step-by-step algorithm before code
  • Add meaningful comments in code
  • Show sample input and expected output
  • Highlight important syntax with colors
Database Query Example
-- Topic: SQL SELECT with WHERE clause
-- Purpose: Filter records based on conditions

-- Table: Students
-- Columns: student_id, name, age, grade, city

-- Query: Find all students with age > 16 and grade = 'A'
SELECT student_id, name, age, grade, city
FROM Students
WHERE age > 16 AND grade = 'A'
ORDER BY name;

-- Sample Output:
-- student_id | name    | age | grade | city
-- 101        | Alice   | 17  | A     | Delhi
-- 103        | Charlie | 18  | A     | Mumbai

1. Introduction to Lists

Definition: A list is an ordered collection of items that can store multiple values in a single variable.

Key Features:

  • Ordered (maintains sequence)
  • Mutable (can be changed)
  • Allows duplicates
  • Can store different data types
2. Creating Lists
# Different ways to create lists
empty_list = []
numbers = [1, 2, 3, 4, 5]
mixed_list = [1, "hello", 3.14, True]
using_constructor = list([1, 2, 3])
3. Common List Methods
Method Purpose Example
append() Add item to end list.append(item)
insert() Add item at position list.insert(index, item)
remove() Remove first occurrence list.remove(item)
Important Notes:
  • List indexing starts from 0
  • Negative indexing starts from -1 (last element)
  • Lists are mutable, so original list changes with methods
